# 剧本查阅移动端 v1.3

这是“剧本管理工作台”的手机 / iPad 查阅端。

## v1.3 完整视觉升级

- 重构为轻专业文档视觉：暖灰白底、深蓝主色、少量金黄强调。
- 统一线性图标，重做顶部栏、底部导航与 iPad 侧栏。
- 原文页调整为真正的阅读器版式，弱化边框与后台感。
- 项目抽屉、角色、场次、素材列表统一信息层级与选中样式。
- 思维导图改为轻节点、细连线与悬浮工具条。
- iPad 横屏采用更完整的双栏工作区。
- 保留 v1.2 的输入框焦点修复、默认原文展示和左侧项目导航。

## 数据安全

导入的 JSON 只保存在当前设备、当前浏览器的 IndexedDB 中，不会自动上传到 GitHub 或其他服务器。请勿把剧本 JSON 或备份数据上传到公开仓库。
