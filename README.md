# 剧本管理移动端 v1.2

供 iPhone、iPad 与桌面浏览器使用的本地剧本查阅工具。

## v1.2 更新

- 剧本原文无需搜索，打开后默认直接显示。
- 修复手机输入一个字后键盘消失的问题。
- 搜索、角色、场景关键词、素材搜索均保持输入焦点。
- 新增可收起的“项目 → 公共文档 / 剧集 → 版本”导航抽屉。
- 选择项目、剧集或版本后，导航自动完全收起。
- 当前选择范围同步作用于原文、角色、场次、导图和素材模块。
- 更新离线缓存策略，GitHub Pages 发布新版后更容易及时刷新。

## 数据安全

网页程序部署在 GitHub Pages；用户手动导入的 JSON 数据仅保存在当前设备浏览器的 IndexedDB 中，不会自动上传到 GitHub。

请勿把剧本 JSON、备份文件或剧本原文上传到公开仓库。
