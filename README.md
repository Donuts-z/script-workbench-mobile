# 剧本管理移动端 v1.5

本版按电脑端视觉语言重新统一移动端：鲜暖黄色、黑色文字、圆润平面组件，并采用“纸张标题式”顶部。

## 本次更新

- 顶部改为方案 C：项目名黑字、当前集/版本黄字、黄色短线标记；
- 去除深蓝色文字，日间模式全部回归黑灰字；
- 去除渐变、高光和立体凸起，选中态只用纯色、字重和黄色短线；
- 黄色提高饱和度，与电脑端更一致；
- 统一圆角、筛选控件高度、字号与垂直对齐；
- 原文场头统一为 `第6集 · 第1场 · 西项 / 边境 / 军营 · 夜 · 外`；
- 场次、素材、导图与项目抽屉统一为暖黄＋黑字体系；
- 修复手机、iPad 与桌面浏览器底部异常空白。

剧本 JSON 仍只保存在导入设备的浏览器 IndexedDB 中，不会上传到 GitHub。
