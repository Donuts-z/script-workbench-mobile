# 剧本管理移动端 v1.1

## 包内文件

- `index.html`：移动端网页应用
- `manifest.webmanifest`：添加到主屏幕后使用的应用信息
- `sw.js`：离线缓存
- `.nojekyll`：让 GitHub Pages 直接发布静态文件
- `icons/`：方案 C（放大镜＋文本线条）图标

## 新增功能

- 场景思维导图
- 单指拖动画布
- 双指缩放（50%—180%）
- 点击节点展开/折叠
- 100% 复位和“适应”画布
- Mac/iPad 触控板双指平移、捏合缩放
- 项目或单集范围切换

## 数据安全

网页代码可以公开发布，但从电脑版导出的 JSON 不要上传到 GitHub。JSON 由用户在 Safari 中手动选择，内容只写入当前设备、当前浏览器的 IndexedDB。

## 更新数据

在电脑版重新“导出移动端数据”，然后在移动端设置里点击“更新数据”，选择新的 JSON 即可。无需重新发布网页。
